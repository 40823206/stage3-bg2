bg9產品展示

第一版

我們發現產品上有些許地方需要進一步去改進，讓它加強切割風阻的能

力，讓車體可以更加穩定的騎乘。例如在面風那側加上導圓角，讓氣流

可以更順的流過。會這樣想是因為之前的設計是平整的面，跟圓角來比

會不於去切割氣流，而且市面上的車也大多是做成流線型的樣子，專門

用減少阻力的產生。

第二版

所以我們將面風處那側加上圓角，讓它可以更加快速的把氣流給分散掉

CoppeliaSim碰撞&動態

"碰撞"屬性
意思即為開啟狀態下，物體會和空間中其他物體發生碰撞，從而模擬實際物理作用狀態。
例:關閉時物體會直接穿過地面，開啟後物體會停於地面上。

碰撞打勾設定解釋:

二物體"無"父子關係的物體，使用"下排"勾勾作設定，假設在相同位置都有勾勾的話，它們就會發生碰撞；假設在每一個位置上，都不會同時出現勾勾的話，他們就不會發生碰撞。
二物體"有"父子關係的物體，使用"上排"勾勾作設定，假設在相同位置都有勾勾的話，它們就會發生碰撞；假設在每一個位置上，都不會同時出現勾勾的話，他們就不會發生碰撞。

"動態"屬性
意思即為關閉狀態下，物體會被視為與世界坐標系固連的關係。
例:關閉時可懸浮於空間之中，不受其他物體碰撞而移動；開啟後會受到向下的重力作用&會受到和其它物件碰撞而產生不同方向的力。

-------------------------------------------------------------------------------------------

"開啟碰撞、動態屬性"

◾在物體圖示點2下左鍵，開啟屬性對話框

◾點Show dynamic properties dialog

◾點Body is respondable(開啟碰撞)，框中勾勾用來做碰撞設定

◾點Body is dynamic(開啟動態)

◾實際操作影片

CoppeliaSim物體顏色、透明度

適當調整各物體顏色、透明度，可易於查看複雜結合件

-------------------------------------------------------------------------------------------

"更改顏色"

◾在物體圖示點2下左鍵，開啟屬性對話框

◾點Adjust oolor

◾點Ambient/diffuse component

◾拖動框中6列數值可調整物體顏色

◾實際操作影片

-------------------------------------------------------------------------------------------

"更改透明度"

◾在物體圖示點2下左鍵，開啟屬性對話框

◾點Adjust oolor

◾將選項Opacity開啟，輸入數值即可更改透明度

◾實際操作影片
